============================================================
ドキュメントクラスの設定
============================================================

LaTeXのドキュメントクラスを指定します。
LaTeXの章の :doc:`latex-document` で書いたようなクラスを指定しておきます。

この時 ``latex_documents`` はデフォルトのままにしておき、
``latex_docclass`` を変更します。

.. code-block:: python

    latex_docclass = {'manual' : 'jsbook'}
