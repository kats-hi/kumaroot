応用編
==================================================

プログラミングをしていると、
「ちょっと変更を加えてテストしてみたい、
だけど上手くいかなかったらすぐに戻したい」
ということは、よくあることです。

その際、まず元ファイルのコピーをとって、
バックアップファイルを作るのですが、
一時的なものだと思って **ver01** とかにしがちです。


Gitではファイルのバックアップを作成する代わりに、
**ブランチ** を作成します。
